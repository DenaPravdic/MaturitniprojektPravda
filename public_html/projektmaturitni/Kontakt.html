Popis maturitního projektu

1. Hlavní stránka

  • Popis: Tato stránka je hlavním vstupním bodem pro všechny návštěvníky webu. Poskytuje základní informace o firmě, jejích službách a dalších stránkách webu. Stránka obsahuje navigační menu pro snadný přístup k jednotlivým sekcím.
  • Výhody: Hlavní stránka slouží jako orientační bod, který zajišťuje návštěvníkům snadný přístup ke všem důležitým informacím. Navigace je intuitivní, což znamená, že každý návštěvník může rychle najít to, co hledá, bez zbytečných kroků.
  • Kód: Používáme semantický HTML kód pro správné strukturování obsahu, což zajišťuje dobrou dostupnost a indexování ve vyhledávačích. K dispozici je jednoduché menu, které umožňuje přístup k dalším stránkám, jako je rezervace nebo kontakt.

2. Rezervace termínu

  • Popis: Na této stránce si mohou uživatelé online zarezervovat termín pro výměnu pneumatik. Tento proces je velmi jednoduchý a uživatelé vyplní formulář s údaji o vozidle, výběru služby a dostupném termínu.
  • Výhody: Výhodou této stránky je její automatizace. Uživatelé se nemusí telefonicky domlouvat na termínu, což šetří čas. Online systém umožňuje uživatelům vybrat si termín, který jim vyhovuje, a rezervace je potvrzena ihned po odeslání formuláře.
  • Vlastnosti: Na stránce je implementován jednoduchý rezervační formulář, který obsahuje políčka pro zadání údajů o vozidle, preferovanou službu (např. přezutí pneumatik) a datum a čas termínu. Tento formulář je odesílán na server, kde jsou data automaticky zpracována.
  • Význam kódu: Využíváme HTML formulář s metodou POST pro bezpečné odesílání dat. JavaScript je používán pro validaci formuláře na straně klienta a pro zobrazení upozornění po úspěšném odeslání.

3. Registrace / Přihlášení

  • Popis: Tento formulář slouží pro registraci nových uživatelů a přihlášení těch stávajících. Registrace je nezbytná pro sledování a správu rezervací. Uživatelské účty poskytují přístup k historickým údajům o službách a termínech.
  • Výhody: Uživatelé mohou spravovat své rezervace, upravovat osobní údaje a zajišťovat si nové termíny. Tento systém poskytuje bezpečný a šifrovaný přístup k jejich osobním údajům a předchozím objednávkám.
  • Vlastnosti: Tento systém používá HTML formulář pro přihlášení a registraci. Hesla jsou šifrována a přenos dat mezi uživatelem a serverem je zabezpečen SSL šifrováním.
  • Význam kódu: Kód obsahuje HTML pro formuláře a CSS pro stylování. JavaScript je použit pro zajištění bezpečnosti (např. kontrola silného hesla) a pro zobrazení chybových zpráv při nesprávném přihlášení nebo registraci.

4. Kontakt

  • Popis: Tato stránka poskytuje veškeré kontaktní údaje pro návštěvníky, kteří si přejí kontaktovat firmu. Zobrazuje adresu, telefonní číslo, e-mail a otevírací dobu. Může také obsahovat formulář pro zaslání zprávy přímo ze stránky.
  • Výhody: Poskytuje všechny potřebné informace na jednom místě. Návštěvníci mohou snadno kontaktovat firmu jak telefonicky, tak e-mailem nebo přes kontaktní formulář.
  • Vlastnosti: Stránka používá jednoduchý HTML formulář pro odesílání zpráv. Obsahuje také Google Mapu pro lepší vizualizaci lokace firmy.
  • Význam kódu: Používá se základní HTML kód pro zobrazení kontaktních údajů, formulář a mapu. Pro validaci a odesílání formuláře je použit PHP kód na serveru.
